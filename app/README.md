# HoopMatch

**HoopMatch** — это мобильное Android-приложение для поиска стритбольных и баскетбольных матчей, а также игроков. Создано в рамках дипломного проекта.

## Цель MVP
Предоставить пользователям удобный интерфейс для:
- Поиска актуальных баскетбольных игр на уличных площадках и в залах.
- Создания собственных матчей с указанием уровня игры, времени и количества участников.
- Просмотра профилей организаторов.

## Технологический стек
- **Язык:** Kotlin
- **Архитектура UI:** Activity + ViewBinding
- **Верстка:** XML (ConstraintLayout, Material Design 3)
- **Хранение данных:** In-memory (локальные коллекции для демонстрации MVP)
- **Списки:** RecyclerView

## Основные экраны (User Flow)
1. **LoginScreen:** Простая авторизация пользователя по имени.
2. **MainScreen:** Лента матчей, фильтрация (Все матчи / Мои матчи), поиск.
3. **CreateMatchScreen:** Форма создания новой игры.
4. **MatchDetailScreen:** Подробная информация об игре, кнопка "Присоединиться".
5. **ProfileScreen:** Карточка организатора со статистикой.
